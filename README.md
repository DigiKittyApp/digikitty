# DigiKitty

Web app designed to help cat owners monitor, understand, and improve their cat's long-term health through centralized tracking, health insights, and preventative care recommendations.

## What is included

- React/Vite mobile app shell
- Cat profile selector
- Health analytics summary
- Medical document vault section
- Basic Node.js API stub at `server/index.js`

## Run locally

```bash
npm install
npm run dev
```

Optional API stub:

```bash
npm run server
```

## Connect to GitHub

From this folder:

```bash
git init
git add .
git commit -m "Create DigiKitty app outline"
git branch -M main
git remote add origin https://github.com/DigiKittyApp/digikitty.git
git push -u origin main
```
